# 🚀 Cursor UI Clone (HTML & CSS Only)

A clean and modern **Cursor UI Clone** built using only **HTML5 and CSS3**.

This project recreates the interface design of Cursor without using JavaScript or any CSS frameworks. The main goal was to practice layout structuring, styling techniques, and responsive design using pure CSS.

---

## 📌 Project Overview

This UI clone includes:

- 🎨 Clean and modern layout
- 📐 Responsive design
- 🧱 Flexbox & CSS Grid
- ✨ Hover effects
- 💡 Semantic HTML structure
- 🎯 Pure CSS styling (No frameworks)

---

## 🛠️ Built With

- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  

No JavaScript. No Bootstrap. No Tailwind.

---

## 💻 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/MohdHuzaifaa/Cursor-clone
   ```

2. Navigate into the project folder:
   ```bash
   cd cursor-ui-clone
   ```

3. Open `index.html` in your browser.
